# PassVault — Менеджер смены паролей

🔐 Локальное зашифрованное хранилище для управления паролями с автоматизированной сменой паролей на 24 сервисах.

## Возможности

- **AES-256-CBC шифрование** — все данные зашифрованы мастер-паролем
- **PBKDF2 деривация ключа** — 100 000 итераций для защиты от брутфорса
- **24 сервиса** — Google, Yandex, VK, Telegram, GitHub, Discord и другие
- **10 категорий** — Почта, Соцсети, Мессенджеры, Разработка, Облако, Работа, Финансы, Развлечения, Игры, Маркетплейсы
- **Генератор паролей** — настраиваемая длина, символы, мнемоника
- **Мастер смены паролей** — пошаговый wizard для смены пароля на сервисе
- **Тёмная/светлая тема** — автоматическое переключение
- **Локальное хранение** — данные не покидают устройство

## Технологии

### Web-приложение
- **Next.js 16** — App Router, React 19
- **TypeScript** — полная типизация
- **Tailwind CSS 4** — стилизация
- **shadcn/ui** — компоненты интерфейса
- **Prisma + SQLite** — база данных
- **Zustand** — управление состоянием
- **CryptoJS** — шифрование

### Мобильное приложение (Android)
- **Capacitor** — обёртка WebView
- **Полноценное SPA** — работает офлайн
- **localStorage** — локальное хранение
- **4 МБ APK** — минимальный размер

## Установка

### Web-приложение

```bash
# Установка зависимостей
npm install

# Инициализация базы данных
npx prisma db push

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

### Мобильное приложение (Android)

```bash
cd mobile

# Установка зависимостей
npm install

# Добавление Android платформы
npx cap add android

# Сборка APK
cd android && ./gradlew assembleDebug

# APK будет в mobile/android/app/build/outputs/apk/debug/app-debug.apk
```

## Безопасность

- Мастер-пароль **не хранится** — используется только для деривации ключа
- Ключ шифрования генерируется через **PBKDF2** (100 000 итераций, SHA-256)
- Данные шифруются **AES-256-CBC** с уникальным IV для каждой записи
- Все операции выполняются **локально** — данные не передаются на сервер
- Автоблокировка при неактивности
