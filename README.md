# PassVault — Менеджер смены паролей

🔐 Локальное зашифрованное хранилище для управления паролями с автоматизированной сменой паролей на 24 сервисах.

## Возможности

- **AES-256-GCM шифрование** — аутентифицированное шифрование через Web Crypto API
- **PBKDF2 деривация ключа** — 600 000 итераций (OWASP/NIST рекомендация) для защиты от брутфорса
- **Невыделяемые ключи** — мастер-ключ хранится как CryptoKey (non-extractable), невозможно извлечь из памяти
- **24 сервиса** — Google, Yandex, VK, Telegram, GitHub, Discord и другие
- **10 категорий** — Почта, Соцсети, Мессенджеры, Разработка, Облако, Работа, Финансы, Развлечения, Игры, Маркетплейсы
- **Генератор паролей** — настраиваемая длина, символы, криптографически безопасный ГСЧ
- **Мастер смены паролей** — пошаговый wizard для смены пароля на сервисе
- **Тёмная/светлая тема** — автоматическое переключение
- **Локальное хранение** — данные не покидают устройство
- **Защита от брутфорса** — экспоненциальная блокировка после неверных попыток, максимум 10 попыток
- **Автоочистка буфера обмена** — скопированные пароли удаляются через 30 секунд
- **Защита от скриншотов** — FLAG_SECURE на Android предотвращает захват экрана

## Установка

### Web-приложение

```bash
# Установка зависимостей
npm install

# Инициализация базы данных
npx prisma db push

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
npm start
```

### Мобильное приложение (Android)

```bash
cd mobile

# Установка зависимостей
npm install

# Добавление Android платформы
npx cap add android

# Сборка APK
cd android && ./gradlew assembleDebug

# APK будет в mobile/android/app/build/outputs/apk/debug/app-debug.apk
```

## Безопасность

### Шифрование
- **AES-256-GCM** — аутентифицированное шифрование (обеспечивает и конфиденциальность, и целостность данных)
- **PBKDF2-HMAC-SHA256** — деривация ключа с 600 000 итерациями (соответствует рекомендациям OWASP/NIST 2024)
- **Невыделяемые ключи** — мастер-ключ хранится как `CryptoKey` с флагом `extractable: false`
- **Уникальный IV** — криптографически безопасный 12-байтный IV генерируется для каждой операции шифрования
- **Безопасная очистка памяти** — сырые ключевые материалы затираются нулями после импорта

### Аутентификация
- **Мастер-пароль не хранится** — используется только для деривации ключа
- **Constant-time сравнение** — защита от атак по времени при проверке пароля
- **Rate limiting** — экспоненциальная задержка после неверных попыток, максимум 10 попыток, затем 5-минутная блокировка

### Защита на устройстве
- **Автоблокировка** — при неактивности 5 минут
- **Автоочистка буфера** — скопированные пароли удаляются через 30 секунд
- **FLAG_SECURE (Android)** — предотвращает скриншоты и запись экрана
- **Локальное хранение** — данные никогда не покидают устройство

### Миграция
- Автоматическая миграция данных из старого формата (CryptoJS AES-256-CBC + HMAC) в новый (Web Crypto API AES-256-GCM) при первом входе
